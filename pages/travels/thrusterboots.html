Thruster Boots

The Thruster Boots allows the user to make short, quick dashes. It stores up to 3 charges that replenish quickly over time.



These dashes can be used in 8 lateral directions; front, back, left, right, and each of their diagonals.



Jumping can help the user gain more momentum and distance when using Thruster Boots.


While Thruster Boots has little vertical momentum, it can be used to navigate terrain or even dodge enemy attacks.
