  1. Click the download button below to get the latest version of Alphatius 2.

Setup Instructions (for Linux / Mac)

  1. Visit Q-Zandronum's website and download the latest version of Q-Zandronum for your operating system.
  2. Extract the files if necessary to a safe and memorable location.
  3. Download the .pk3 files below (and any relevant patch files) and place them into your Q-Zandronum directory.
  4. Drag and drop all .pk3 files onto the Q-Zandronum program to play single player.
  5. To play multiplayer, Q-Zandronum comes with a server browser named "Doomseeker" in the "Doomseeker" folder.
  6. Make sure Doomseeker is properly configured by pointing it to your Q-Zandronum installation in the configuration menu (F5).
  7. Search for "Alphatius" in the search filters to find a suitable server and join.
